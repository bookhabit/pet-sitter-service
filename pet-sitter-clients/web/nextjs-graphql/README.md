# Next.js + GraphQL Client

Next.js web application consuming the Pet Sitter Service GraphQL API.

## Tech Stack
- Next.js (App Router)
- TypeScript
- GraphQL
- Apollo Client

## Features
- User authentication
- Job listing and management
- Job application system
- Role-based UI (PetOwner/PetSitter)

## Setup
```bash
npm install
npm run dev
```

## API
Connects to `http://localhost:3000/graphql` (GraphQL endpoint)
