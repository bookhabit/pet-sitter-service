# Vanilla JavaScript + GraphQL Client

Vanilla JavaScript web application consuming the Pet Sitter Service GraphQL API.

## Tech Stack
- Vanilla JavaScript
- HTML/CSS
- Fetch API
- GraphQL

## Features
- User authentication
- Job listing and management
- Job application system
- Role-based UI (PetOwner/PetSitter)

## Setup
```bash
# Serve with any static file server
npx serve
```

## API
Connects to `http://localhost:3000/graphql` (GraphQL endpoint)
