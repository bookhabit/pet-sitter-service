# Next.js + REST API Client

Next.js web application consuming the Pet Sitter Service REST API.

## Tech Stack
- Next.js (App Router)
- TypeScript
- REST API

## Features
- User authentication
- Job listing and management
- Job application system
- Role-based UI (PetOwner/PetSitter)

## Setup
```bash
npm install
npm run dev
```

## API
Connects to `http://localhost:3000` (REST endpoints)
