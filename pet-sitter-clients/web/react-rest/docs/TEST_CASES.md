# [SIGN] 회원가입

## SIGN-01 | 페이지 접근

#### SIGN-01-001 비로그인 사용자의 회원가입 페이지 접근
- **GIVEN** 로그인하지 않은 사용자가
- **WHEN** `/signup` 페이지에 접근하면
- **THEN** 회원가입 폼이 표시된다

#### SIGN-01-002 이미 로그인한 사용자의 회원가입 페이지 접근
- **GIVEN** 이미 로그인한 사용자가
- **WHEN** `/signup` 페이지에 접근하면
- **THEN** 회원가입 페이지 대신 다른 페이지로 이동된다

---

## SIGN-02 | 역할 선택

#### SIGN-02-001 PetOwner 역할 선택
- **GIVEN** 회원가입 폼이 표시된 상태에서
- **WHEN** 사용자가 `PetOwner` 버튼을 클릭하면
- **THEN** `PetOwner` 버튼이 선택된 상태로 표시된다

#### SIGN-02-002 PetSitter 역할 선택
- **GIVEN** 회원가입 폼이 표시된 상태에서
- **WHEN** 사용자가 `PetSitter` 버튼을 클릭하면
- **THEN** `PetSitter` 버튼이 선택된 상태로 표시된다

#### SIGN-02-003 역할 변경
- **GIVEN** `PetOwner`를 선택한 상태에서
- **WHEN** 사용자가 `PetSitter` 버튼을 클릭하면
- **THEN** `PetSitter`만 선택된 상태로 변경된다

#### SIGN-02-004 역할 미선택 상태에서 제출
- **GIVEN** 역할을 선택하지 않고 나머지 항목을 모두 올바르게 입력한 상태에서
- **WHEN** 사용자가 회원가입 버튼을 클릭하면
- **THEN** 가입이 진행되지 않고 역할 선택을 요구하는 안내가 표시된다

---

## SIGN-03 | 이름 입력 유효성 검사

- **WHEN** 사용자가 이름을 1자만 입력하면
- **THEN** 오류 메시지가 표시되고 입력 필드가 오류 상태로 표시된다

- **WHEN** 사용자가 이름을 2자 입력하면
- **THEN** 유효성을 통과하고 입력 필드가 정상 상태로 표시된다

#### SIGN-03-003 이름 최대 길이 초과
- **GIVEN** 회원가입 폼이 표시된 상태에서
- **WHEN** 사용자가 이름을 51자 입력하면
- **THEN** 오류 메시지가 표시되고 입력 필드가 오류 상태로 표시된다

#### SIGN-03-004 이름에 숫자 포함
- **GIVEN** 회원가입 폼이 표시된 상태에서
- **WHEN** 사용자가 이름에 숫자를 포함해 입력하면
- **THEN** 오류 메시지가 표시되고 입력 필드가 오류 상태로 표시된다

#### SIGN-03-005 이름을 공백으로만 입력
- **GIVEN** 회원가입 폼이 표시된 상태에서
- **WHEN** 사용자가 이름 필드에 공백만 입력하면
- **THEN** 오류 메시지가 표시되고 입력 필드가 오류 상태로 표시된다

## SIGN-06 | 폼 제출

#### SIGN-06-001 유효하지 않은 입력으로 제출

- **THEN** 서버로 요청이 전송되지 않고 각 항목의 오류 메시지가 표시된다

- **GIVEN** 역할 선택 및 모든 입력 항목이 올바르게 입력된 상태에서
- **WHEN** 사용자가 회원가입 버튼을 클릭하면
- **THEN** 버튼에 로딩 상태가 표시되고 서버로 요청이 전송된다

#### SIGN-06-003 요청 중 중복 제출 방지
- **GIVEN** 회원가입 요청이 진행 중인 상태에서
- **WHEN** 사용자가 다시 회원가입 버튼을 클릭하면
- **THEN** 중복 요청이 전송되지 않는다

# [AUTH] 로그인 / 인증

## AUTH-01 | 페이지 접근

#### AUTH-01-001 비로그인 사용자의 로그인 페이지 접근
- **GIVEN** 로그인하지 않은 사용자가
- **WHEN** `/login` 페이지에 접근하면
- **THEN** 로그인 폼이 표시된다

#### AUTH-01-002 이미 로그인한 사용자의 로그인 페이지 접근
- **GIVEN** 이미 로그인한 사용자가
- **WHEN** `/login` 페이지에 접근하면
- **THEN** 로그인 페이지 대신 다른 페이지로 이동된다

#### AUTH-01-003 회원가입 완료 후 로그인 페이지 진입
- **GIVEN** 회원가입을 막 완료한 사용자가
- **WHEN** 로그인 페이지로 이동되면
- **THEN** 회원가입이 완료되었다는 안내 메시지가 표시된다

#### AUTH-01-004 회원가입 페이지 링크 확인
- **GIVEN** 로그인 폼이 표시된 상태에서
- **WHEN** 페이지를 확인하면
- **THEN** 회원가입 페이지로 이동하는 링크가 표시된다

# [JOB] 구인공고

> **필수 입력 항목 기준**
> - 돌봄 시작 일시 / 종료 일시
> - 돌봄 내용 (5자 이상 500자 이하)
> - 반려동물 (1마리 이상, 최대 10마리) — 각 동물의 이름 / 나이 / 종(Cat·Dog) / 품종
>
> **선택 입력 항목**
> - 주소 / 가격 / 가격 단위(시간당·일당) / 사진

## JOB-01 | 구인공고 등록

### 접근 및 렌더링

#### JOB-01-001 PetSitter의 등록 접근 차단
- **GIVEN** PetSitter 역할로 로그인한 사용자가
- **WHEN** 구인공고 등록 페이지에 접근하면
- **THEN** 접근이 차단되거나 등록 버튼이 표시되지 않는다

#### JOB-01-002 PetOwner의 등록 페이지 접근
- **GIVEN** PetOwner 역할로 로그인한 사용자가
- **WHEN** 구인공고 등록 페이지에 접근하면
- **THEN** 구인공고 등록 폼이 표시된다

---

### 돌봄 일정 유효성

#### JOB-01-003 시작 일시 미입력
- **GIVEN** PetOwner가 등록 폼에서 시작 일시를 입력하지 않은 상태에서
- **WHEN** 등록 버튼을 클릭하면
- **THEN** 시작 일시 항목에 오류 메시지가 표시되고 등록이 진행되지 않는다

#### JOB-01-004 종료 일시 미입력
- **GIVEN** PetOwner가 등록 폼에서 종료 일시를 입력하지 않은 상태에서
- **WHEN** 등록 버튼을 클릭하면
- **THEN** 종료 일시 항목에 오류 메시지가 표시되고 등록이 진행되지 않는다

#### JOB-01-005 종료 일시가 시작 일시보다 빠른 경우
- **GIVEN** PetOwner가 종료 일시를 시작 일시보다 이전으로 입력한 상태에서
- **WHEN** 등록 버튼을 클릭하면

- **THEN** 이미 사용 중인 이메일이라는 안내 메시지가 표시된다

#### USER-03-006 비밀번호 최소 길이 미달 (7자 이하)

- **THEN** 비밀번호 항목에 오류 메시지가 표시되고 저장이 진행되지 않는다

#### USER-03-007 비밀번호 최대 길이 초과 (21자 이상)
- **GIVEN** 프로필 수정 폼에서 비밀번호를 21자 이상 입력한 상태에서

# [JOBAPP] 구인공고 지원

> **지원 가능 역할**: PetSitter 전용
> **상태값**: applying(지원 중) → approved(승인) / rejected(거절)

## JOBAPP-01 | 지원

#### JOBAPP-01-001 PetOwner의 지원 버튼 미표시
- **GIVEN** PetOwner 역할로 로그인한 사용자가 구인공고 상세 페이지를 보고 있는 상태에서
- **WHEN** 페이지를 확인하면
- **THEN** 지원하기 버튼이 표시되지 않는다

#### JOBAPP-01-002 PetSitter의 구인공고 지원
- **GIVEN** PetSitter 역할로 로그인한 사용자가 아직 지원하지 않은 공고의 상세 페이지를 보고 있는 상태에서
- **WHEN** 지원하기 버튼을 클릭하면
- **THEN** 지원이 완료되고 해당 공고에 지원 완료 상태가 표시된다

#### JOBAPP-01-003 이미 지원한 공고에 중복 지원 시도
- **GIVEN** PetSitter가 이미 지원한 공고의 상세 페이지를 보고 있는 상태에서
- **WHEN** 페이지를 확인하면
- **THEN** 지원하기 버튼 대신 이미 지원 완료 상태가 표시된다

#### JOBAPP-01-004 지원 중 로딩 상태
- **GIVEN** PetSitter가 지원하기 버튼을 클릭한 상태에서
- **WHEN** 지원 요청이 처리되는 동안
- **THEN** 버튼에 로딩 상태가 표시되고 중복 클릭이 방지된다

#### JOBAPP-01-005 네트워크 오류 발생 시 지원 실패
- **GIVEN** 네트워크 연결이 불안정한 상태에서 PetSitter가 지원을 시도하면
- **WHEN** 요청이 실패하면
- **THEN** 오류 메시지가 표시되고 지원 이전 상태로 유지된다

## JOBAPP-02 | 지원자 목록 조회

#### JOBAPP-02-001 공고 작성자의 지원자 목록 확인

- **THEN** 지원한 PetSitter들의 정보와 지원 상태가 표시된다

#### JOBAPP-02-002 지원자가 없는 공고의 목록 조회
- **GIVEN** 지원자가 한 명도 없는 공고의 작성자가 상세 페이지를 보고 있는 상태에서
- **WHEN** 지원자 목록 영역을 확인하면
- **THEN** 지원자가 없다는 빈 상태 안내가 표시된다

#### JOBAPP-02-003 공고 작성자가 아닌 사용자의 지원자 목록 미표시
- **GIVEN** 해당 공고를 등록하지 않은 사용자가 공고 상세 페이지를 보고 있는 상태에서
- **WHEN** 페이지를 확인하면
- **THEN** 다른 지원자의 지원 정보가 표시되지 않는다

## JOBAPP-03 | 지원 상태 변경 (승인 / 거절)

#### JOBAPP-03-001 공고 작성자의 지원 승인
- **GIVEN** 구인공고를 등록한 PetOwner가 지원자 목록에서 특정 지원자를 보고 있는 상태에서
- **WHEN** 승인 버튼을 클릭하면
- **THEN** 해당 지원자의 상태가 승인됨으로 변경되어 표시된다

#### JOBAPP-03-002 공고 작성자의 지원 거절
- **GIVEN** 구인공고를 등록한 PetOwner가 지원자 목록에서 특정 지원자를 보고 있는 상태에서
- **WHEN** 거절 버튼을 클릭하면
- **THEN** 해당 지원자의 상태가 거절됨으로 변경되어 표시된다

#### JOBAPP-03-003 공고 작성자가 아닌 사용자의 상태 변경 버튼 미표시
- **GIVEN** 해당 공고를 등록하지 않은 사용자가 공고 상세 페이지를 보고 있는 상태에서
- **WHEN** 페이지를 확인하면
- **THEN** 지원 상태를 변경하는 버튼이 표시되지 않는다

#### JOBAPP-03-004 PetSitter가 본인 지원 상태 확인
- **GIVEN** 지원한 PetSitter가 해당 공고 상세 페이지를 보고 있는 상태에서
- **WHEN** 페이지를 확인하면
- **THEN** 본인의 지원 상태(지원 중 / 승인됨 / 거절됨)가 표시된다

- **WHEN** 페이지를 확인하면
- **THEN** 리뷰 작성 버튼 또는 입력 폼이 표시된다

#### REVIEW-01-002 승인된 지원이 없는 공고에서 리뷰 작성 불가
- **GIVEN** 지원이 아직 승인되지 않은 공고를 보고 있는 상태에서
- **WHEN** 페이지를 확인하면
- **THEN** 리뷰 작성 버튼이 표시되지 않거나 비활성화된다

#### REVIEW-01-003 별점 미입력
- **GIVEN** 리뷰 작성 폼에서 별점을 선택하지 않은 상태에서
- **WHEN** 등록 버튼을 클릭하면
- **THEN** 별점 항목에 오류 메시지가 표시되고 등록이 진행되지 않는다

#### REVIEW-01-004 별점 범위 외 값 입력 (1 미만 또는 5 초과)
- **GIVEN** 리뷰 작성 폼에서 1~5 범위를 벗어난 별점을 입력한 상태에서
- **WHEN** 등록 버튼을 클릭하면
- **THEN** 별점 항목에 오류 메시지가 표시되고 등록이 진행되지 않는다

#### REVIEW-01-005 별점만 입력하고 코멘트 없이 등록
- **GIVEN** 리뷰 작성 폼에서 별점(1~5)만 선택하고 코멘트를 비워둔 상태에서
- **WHEN** 등록 버튼을 클릭하면
- **THEN** 리뷰가 정상적으로 등록된다

#### REVIEW-01-006 별점과 코멘트 모두 입력 후 등록
- **GIVEN** 리뷰 작성 폼에서 별점과 코멘트를 모두 입력한 상태에서
- **WHEN** 등록 버튼을 클릭하면
- **THEN** 리뷰가 등록되고 해당 내용이 리뷰 목록에 표시된다

#### REVIEW-01-007 동일 공고에 리뷰 중복 작성 시도

- **WHEN** 페이지를 확인하면
- **THEN** 리뷰 작성 폼이 표시되지 않거나 이미 리뷰가 작성되었다는 안내가 표시된다

## REVIEW-03 | 리뷰 삭제

#### REVIEW-03-001 리뷰 작성자에게만 삭제 버튼 표시
- **GIVEN** 리뷰 목록이 표시된 상태에서
- **WHEN** 페이지를 확인하면
- **THEN** 본인이 작성한 리뷰에만 삭제 버튼이 표시된다

#### REVIEW-03-002 리뷰 작성자의 삭제
- **GIVEN** 본인이 작성한 리뷰의 삭제 버튼이 표시된 상태에서
- **WHEN** 삭제 버튼을 클릭하면
- **THEN** 리뷰가 삭제되고 해당 리뷰가 목록에서 제거된다

# [FAV] 즐겨찾기

> **사용 가능 역할**: PetSitter 전용
> **동작 방식**: 동일 공고 클릭 시 추가 / 제거 토글

---

## FAV-01 | 즐겨찾기 추가

#### FAV-01-001 PetOwner에게 즐겨찾기 버튼 미표시
- **GIVEN** PetOwner 역할로 로그인한 사용자가 구인공고 상세 페이지를 보고 있는 상태에서
- **WHEN** 페이지를 확인하면
- **THEN** 즐겨찾기 버튼이 표시되지 않는다

#### FAV-01-002 PetSitter가 즐겨찾기에 추가
- **GIVEN** PetSitter 역할로 로그인한 사용자가 즐겨찾기하지 않은 공고의 상세 페이지를 보고 있는 상태에서
- **WHEN** 즐겨찾기 버튼을 클릭하면
- **THEN** 즐겨찾기에 추가되고 버튼이 활성 상태로 변경된다

#### FAV-01-003 이미 즐겨찾기 된 공고 재클릭 (토글 제거)
- **GIVEN** PetSitter가 이미 즐겨찾기에 추가한 공고의 상세 페이지를 보고 있는 상태에서
- **WHEN** 활성화된 즐겨찾기 버튼을 다시 클릭하면
- **THEN** 즐겨찾기에서 제거되고 버튼이 비활성 상태로 변경된다

---

## FAV-02 | 즐겨찾기 목록 조회

#### FAV-02-001 즐겨찾기 목록 표시
- **GIVEN** PetSitter 역할로 로그인한 사용자가
- **WHEN** 즐겨찾기 목록 페이지에 접근하면
- **THEN** 즐겨찾기에 추가한 구인공고 목록이 최근 추가 순으로 표시된다

#### FAV-02-002 즐겨찾기가 없는 경우
- **GIVEN** PetSitter가 즐겨찾기에 추가한 공고가 하나도 없는 상태에서
- **WHEN** 즐겨찾기 목록 페이지에 접근하면
- **THEN** 즐겨찾기가 없다는 빈 상태 안내가 표시된다

#### FAV-02-003 PetOwner의 즐겨찾기 목록 페이지 접근 차단
- **GIVEN** PetOwner 역할로 로그인한 사용자가
- **WHEN** 즐겨찾기 목록 페이지에 접근하면
- **THEN** 접근이 차단되거나 해당 메뉴가 표시되지 않는다

---

## FAV-03 | 즐겨찾기 제거

#### FAV-03-001 즐겨찾기 목록에서 제거
- **GIVEN** PetSitter가 즐겨찾기 목록 페이지를 보고 있는 상태에서
- **WHEN** 특정 공고의 즐겨찾기 제거 버튼을 클릭하면
- **THEN** 해당 공고가 즐겨찾기 목록에서 제거된다

#### FAV-03-002 공고 상세 페이지에서 즐겨찾기 제거
- **GIVEN** PetSitter가 즐겨찾기에 추가된 공고의 상세 페이지를 보고 있는 상태에서
- **WHEN** 활성화된 즐겨찾기 버튼을 클릭하면
- **THEN** 즐겨찾기에서 제거되고 즐겨찾기 목록에서도 해당 공고가 사라진다
