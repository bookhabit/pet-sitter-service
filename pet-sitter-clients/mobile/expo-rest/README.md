# Expo (React Native) + REST API Client

Expo mobile application consuming the Pet Sitter Service REST API.

## Tech Stack
- Expo
- React Native
- TypeScript
- REST API

## Features
- User authentication
- Job listing and management
- Job application system
- Role-based UI (PetOwner/PetSitter)

## Setup
```bash
npm install
npx expo start
```

## API
Connects to `http://localhost:3000` (REST endpoints)

## Platform Support
- iOS
- Android
- Web (PWA)
