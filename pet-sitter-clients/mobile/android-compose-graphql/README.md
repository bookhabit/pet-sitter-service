# Android Jetpack Compose + GraphQL Client

Native Android application consuming the Pet Sitter Service GraphQL API.

## Tech Stack
- Kotlin
- Jetpack Compose
- Apollo Android
- GraphQL

## Features
- User authentication
- Job listing and management
- Job application system
- Role-based UI (PetOwner/PetSitter)

## Setup
```bash
# Open in Android Studio
# Build and run on emulator or device
```

## API
Connects to `http://10.0.2.2:3000/graphql` (GraphQL endpoint - emulator)

## Requirements
- Android Studio
- Android SDK 24+
