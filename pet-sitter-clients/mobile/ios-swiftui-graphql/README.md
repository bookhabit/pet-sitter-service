# iOS SwiftUI + GraphQL Client

Native iOS application consuming the Pet Sitter Service GraphQL API.

## Tech Stack
- Swift
- SwiftUI
- Apollo iOS
- GraphQL

## Features
- User authentication
- Job listing and management
- Job application system
- Role-based UI (PetOwner/PetSitter)

## Setup
```bash
# Open in Xcode
# Build and run on simulator or device
```

## API
Connects to `http://localhost:3000/graphql` (GraphQL endpoint)

## Requirements
- Xcode 15+
- iOS 17+
- macOS development environment
