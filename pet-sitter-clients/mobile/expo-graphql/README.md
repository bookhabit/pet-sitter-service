# Expo (React Native) + GraphQL Client

Expo mobile application consuming the Pet Sitter Service GraphQL API.

## Tech Stack
- Expo
- React Native
- TypeScript
- GraphQL
- Apollo Client

## Features
- User authentication
- Job listing and management
- Job application system
- Role-based UI (PetOwner/PetSitter)

## Setup
```bash
npm install
npx expo start
```

## API
Connects to `http://localhost:3000/graphql` (GraphQL endpoint)

## Platform Support
- iOS
- Android
- Web (PWA)
