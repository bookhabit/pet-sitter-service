# Android Jetpack Compose + REST API Client

Native Android application consuming the Pet Sitter Service REST API.

## Tech Stack
- Kotlin
- Jetpack Compose
- Retrofit
- REST API

## Features
- User authentication
- Job listing and management
- Job application system
- Role-based UI (PetOwner/PetSitter)

## Setup
```bash
# Open in Android Studio
# Build and run on emulator or device
```

## API
Connects to `http://10.0.2.2:3000` (REST endpoints - emulator)

## Requirements
- Android Studio
- Android SDK 24+
