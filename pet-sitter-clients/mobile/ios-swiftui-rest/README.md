# iOS SwiftUI + REST API Client

Native iOS application consuming the Pet Sitter Service REST API.

## Tech Stack
- Swift
- SwiftUI
- URLSession
- REST API

## Features
- User authentication
- Job listing and management
- Job application system
- Role-based UI (PetOwner/PetSitter)

## Setup
```bash
# Open in Xcode
# Build and run on simulator or device
```

## API
Connects to `http://localhost:3000` (REST endpoints)

## Requirements
- Xcode 15+
- iOS 17+
- macOS development environment
